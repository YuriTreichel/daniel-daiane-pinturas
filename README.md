Landing page profissional para a **Daniel & Daiane Pinturas e Restaurações**, uma empresa familiar com mais de 60 anos de tradição em Bento Gonçalves e região, especializada em acabamentos finos e restaurações.

## 🚀 Tecnologias

O projeto foi desenvolvido utilizando as tecnologias mais modernas do mercado:

- **React 19**: Core do projeto para uma interface reativa e performática.
- **Vite**: Ferramenta de build e servidor de desenvolvimento de última geração.
- **Tailwind CSS 4**: Estilização baseada em utilitários para um design customizado e leve.
- **Motion (Framer Motion)**: Implementação de animações fluidas e experiências interativas.
- **Lucide React**: Ícones vetoriais modernos e consistentes.
- **TypeScript**: Garantia de código robusto e livre de erros comuns.

## ✨ Funcionalidades Principais

- **Design Premium & Responsivo**: Experiência visual impecável em qualquer tamanho de tela.
- **Conversão via WhatsApp**: Botões estratégicos e formulários integrados para conversão direta.
- **Galeria de Projetos Interativa**: Portfolio dinâmico com suporte a vídeos e imagens em modal (popup).
- **Legado Familiar (Documentário)**: Seção exclusiva em vídeo dedicada à história e valores da família.
- **Diferencial Técnico**: Foco na exposição do ferramental profissional de última geração.
- **Navegação Inteligente**: Links de menu organizados de acordo com a jornada do usuário na página.

## 🛠️ Execução Local

**Pré-requisitos:** Node.js instalado.

1. Instale as dependências do projeto:
   ```bash
   npm install
   ```
2. Inicie o ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
3. O projeto estará disponível em `http://localhost:3000`.
